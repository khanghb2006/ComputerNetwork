Project Overview

The RATUS

This website acts as the final-term project for the Computer Networks course at Ho Chi Minh Univerity of Science (24CNTN). It implements a Remote Administration System using a Client–Server architecture, where the Client acts as a controlled agent and the Server serves as a centralized administration console. The project emphasizes socket-based communication, real-time data exchange, and practical application of core networking principles.
